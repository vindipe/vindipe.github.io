# AGENTS.md

## Project
Personal website built with Astro, Tailwind CSS, DaisyUI and GitHub Pages.

## Commands
- Install: `pnpm install`
- Dev: `pnpm run dev`
- Build: `pnpm run build`

## Scope
The first public version includes only:
- Home
- Projects
- Publications
- CV
- Contact

Do not reintroduce:
- Blog
- Store
- Services
- RSS
- demo template content
- early-stage project names intentionally removed

## Editing rules
- Inspect files before editing.
- Preserve existing layout and content unless explicitly asked.
- Prefer minimal diffs.
- Run `pnpm run build` after code changes.
- Show changed files and summarize changes.
